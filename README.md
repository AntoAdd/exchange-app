# Exchange App

This repository contains a web application designed to facilitate product exchange offers.

## Features

* **Offer Creation:** Users can create offers by specifying a product they wish to exchange.
* **Counteroffers:** Other users can respond to offers with counteroffers, proposing one or more of their own products in exchange.
* **Product Exchange Management:** Functionality to manage and display offers and counteroffers.
* **User Interaction:** Allows users to view and interact with existing exchange offers.

## Technologies Used

* **Frontend:**
    * **React:** For building the interactive user interface.
    * **Bootstrap:** For styling the application.
* **Backend:**
    * **Java with Spring Boot:** For building the RESTful API and handling backend logic.
    * **PostgreSQL:** For persistent storage of offers, counteroffers, and user data.


## Future Improvements
**Project is under active development and not feature complete.**

* **Enhanced UI/UX:** Improve the user interface and user experience.
* **Image Uploads:** Allow users to upload images of their products.
* **Notifications:** Implement notifications for new offers and counteroffers.
* **Search and Filtering:** Add search and filtering capabilities for offers and geographic regions.
